# AquaMind Website Bundle

Ten folder zawiera komplet stron do publikacji na GitHub Pages:

- `index.html`
- `privacy.html`
- `terms.html`
- `delete-account.html`
- `admin/index.html`

## Publikacja (GitHub Pages)

1. Skopiuj zawartosc tego folderu do repo `aquamind-website`.
2. Upewnij sie, ze pliki sa w katalogu glownym repo strony.
3. Wlacz GitHub Pages (branch: `main`, folder: `/root`).
4. Zweryfikuj URL-e:
   - `https://aquamindlabs.github.io/aquamind-website/`
   - `https://aquamindlabs.github.io/aquamind-website/privacy.html`
   - `https://aquamindlabs.github.io/aquamind-website/terms.html`
   - `https://aquamindlabs.github.io/aquamind-website/delete-account.html`
